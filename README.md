Вот как установить данный пакетнрый менеджер :

1:
git clone https://github.com/masterarbyzpik02-hub/wormcompiller.git
2:
скопируйте код из файла worm,
затем введите в консоль это:
sudo nano /usr/local/bin/worm 
и вставьте скопированый код в данный файл, и сохраните CTRL+S+X . 

Все теперь вы можете использовать данный пакетный менеджер !
